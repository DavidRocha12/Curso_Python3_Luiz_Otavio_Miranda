## Estrutura do Repositório

O conteúdo está organizado da seguinte forma:
- **Exercícios do Curso:** Desafios e projetos práticos propostos no curso de Python do instrutor **Luiz Otávio Miranda**.
- **Exercícios Complementares:** Desafios personalizados gerados com o suporte de Inteligência Artificial para fixação de conceitos.

- **Controle de Versão:** Git e GitHub
